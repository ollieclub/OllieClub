About

Fish memory is a template for memory game, you can custom the dificulty and custom the assets

Getting Started

There is 3 folder you should get

  • Demo : Finished game *
  • Source : Game source
  • Documentation : Documentation about for the game

*Demo & source does not include bgm, read section Installation > change BGM for further information

There is two options you can follow to edit the template

  • Simple Edit If you want to just reskin withouth edit the score or difficulty

  • Advance Edit If you want to edit difficulty or make the game become like animal memory

Simple Edit

You can simply change the assets without edit configuration Go to dirrectory Demo/assets and just replace the image you want to change then you can upload the game to your server

Advance Edit

If you want to change some configuration, like edit difficulty or info use this guide

Requirement:

  • Text Editor (Visual Code, Atom, notepad++ or etc)
  • Node.js

Folder Structure

  • Assets all assets used in the game, you can change audio and images
  • src Source code and configuration for the game
  • dist Exported game for production
  • defs Intellisense for phaser

Installation

If you have node js installed go to your dirrectory in command promp/terminal

Go to path source

#Windows:

  • Go to your project folder then shift+click source folder then > copy as path
  • Search "cmd" then open it
  • Then type "cd" give blank space then right click > paste
cd "E:\game\source"
  • If you save the project other than your windows drive usually C:\ then type your drive letter for example "E:" then hit enter

#OSX:

  • Open terminal
  • type "cd" give blank space
cd "/Users/name/project/source"
  • Drag folder to terminal then press enter

Install project

This steps is same in every OS

  • type "npm install" then wait until finish

Configuration

If you want to change some difficulty or other information go to folder Data in "Source/src/Data"

open Config.json if you want to change game settings

{
    "Difficulty": { 
        "Easy": { // Difficulty Name
            "chance": 5, // Chance before show result screen
            "bonusNextLvlChance": 1, // Bonus change after complete the memory
            "totalBox": 4, // Total card
            "sizeBox": 200, // Size of the card in px, ex 200x200 px
            "grid": {
                "width": 2, // Total card per column
                "cellWidth": 210, // width gap sizebox + gap per card ( gap 10px )
                "cellHeight": 210, // height gap sizebox + gap per card ( gap 10px )
                "x": 0, // X position of the card
                "y": -50 // Y position of the card
            }
        }
    },
    "Bgm": ["bgm.ogg","bgm.m4a"], // Name of bgm, you also can change to mp3
    "EnableFullscreenButton": true, // Display fullscreen button it's recomended for pc/web
    "Credits": {
        "Logo": { // Logo position
            "x": 0,
            "y": -10
        },
        "Text": { // Text setting in bottom logo 
            "x": 0, // X Text position
            "y": 85, // Y Text position
            "fontSize": 24, // Font size
            "color": "#000000", // Color of text
            "text": "Created by : Kecapi Hejo" // Text in credits
        }
    },
    "DefaultScore": 100, // Score per right answer
    "DelayWrongCard": 1, // Delay before can select another card
    "Animation": {
        "CardTweenSpeed": 200 // Card animation flip speed
    }
}

open Fruits.json if you want to add more variety to cards

To add more variety cards, the card image extension should be PNG

{
    "name": "Orange Half",
    "id": "orange_half"
}.
{// Example new card
    "name": "Grape",
    "id": "grape" // name card in Assets/images/card
}

Publish

After done with changing configuration open command promp/terminal again then go to dirrectory like in steps before, then type

npm run build

Wait until done it will take some time After done the game will be in dist dirrectory, then you can upload to your server

Change BGM

Default bgm folder is Assets/bgm and default name for bgm is "bgm"

If you don't change the configuration bgm folder need 2 files

for bgm to work in most browser you need bgm with aac-m4a and ogg

You can use software in tools section to free convert audio

Audio that used in preview is "Nouvelle Noel" by incompetech
